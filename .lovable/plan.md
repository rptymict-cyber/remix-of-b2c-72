
# 농산물 시세 예측 서비스 — 전면 고도화 계획

정부 과제 제출용으로 모든 버튼/페이지/바텀시트가 동작해야 한다는 점을 전제로, 현 코드베이스(React + Vite + Tailwind + shadcn, 5개 페이지: Index/MarketPrice/AIPrediction/SalesChannel/CropRecommend)에 맞춰 다음 작업을 단계별로 수행합니다.

## 1. 공통 인프라 (전 화면에서 재사용)

- **상태 스토어 추가** `src/store/appStore.ts`
  - 선택 작물(`crop`), 품종(`variety`), 시장(`market`), 단량(`unit`), 출하량(`shipQty`)
  - 사용자 프로필(이름·지역·면적·재배작물·구독플랜·알림설정)
  - localStorage persist → 새 진입 시 온보딩 미완료면 `/onboarding` 강제 라우팅
  - 단일 소스로 묶어 "칩과 데이터 불일치" 재발 방지
- **데이터 카탈로그** `src/data/catalog.ts`
  - 작물: 사과/배추/양파/무/고추/토마토/딸기/감자/마늘/옥수수 (이모지·품종·기준 단량 포함)
  - 시장: 가락/대구북부/부산엄궁/안양/광주서부/강서/수원/청주 (지역·거리 계산용 좌표)
  - 작물×시장×품종 매트릭스 mock 데이터 + 시계열 생성기
- **공통 컴포넌트**
  - `CropChip`, `MarketChip`, `UnitChip` (이모지+라벨 표준화, 한 곳에서 수정 시 전 앱 반영)
  - `Sheet` 래퍼(shadcn Drawer 기반), `KpiCell`, `RankingRow`, `EmptyState`, `SkeletonCard`, `StaleBanner`, `MarketHolidayNotice`
  - `UpgradeModal` (Free → 유료 기능 접근 차단)

## 2. 디자인 시스템 정합

- `src/index.css`/`tailwind.config.ts`에 토큰 정리: primary `#16A34A`, primary-dark `#15803D`, light-green-bg `#F0FDF4`, gray 단계, 상승=red, 하락=blue (한국 관례)
- 카드 라운드 16px / 소형 12px, shadow·padding 토큰 통일
- AppHeader 좌측 메뉴 아이콘을 로고 + 우측 알림/프로필 두 아이콘 구조로 교체, `/notifications`·`/mypage`로 라우팅

## 3. 신규 라우트

| 경로 | 화면 |
|---|---|
| `/onboarding` | 스플래시 → 소개 캐러셀(3) → 이름 → 지역 → 규모 → 작물 선택 → 완료 |
| `/notifications` | 알림 목록 (탭: 전체/가격/예측/시스템) + 빈 상태 |
| `/notification-settings` | 토글/임계값/D-3·D-1 설정 |
| `/mypage` | 프로필·구독 카드·농장정보·알림설정·앱설정·계정 |

`App.tsx`에 라우트 추가, 온보딩 미완료 시 자동 진입.

## 4. 바텀시트(공용, 모든 ▼ 칩에서 호출)

1. **작물 선택** — 내 작물/전체 탭, 검색, 3열 그리드, 다중 선택 시 체크
2. **시장 선택** — 지역 필터 칩, 거리 표시
3. **출하량 입력** — 숫자 입력 + 단위 토글(kg/상자/톤), 면적 기반 자동 계산
4. **시세 필터** — 작물/품종/시장/기간/단위/정렬 통합
5. **판매처 시장 상세** — 거리·영업·예상 단가/물류비/순이익·법인청과·CTA
6. **작물 상세** — 시세·예측·주요 시장·기상 민감도·관련 소식 + 시세/예측 이동

모든 ▼ 클릭은 store 업데이트 → 모든 카드 동시 반영.

## 5. 탭별 보강

- **홈(`/`)**: 운영 컨텍스트 카드 → 내 작물 가로 칩 → 현재 시세 카드(상세→/market) → AI 예측 카드(상세→/prediction) → 빠른 진입 4 CTA. 알림/프로필 헤더 아이콘 활성화.
- **시세(`/market`)**: 필터바 5개 ▼ 모두 바텀시트 연결, KPI 6칸, 듀얼 차트(기간·뷰 토글), 분석 탭 5개(종합/시장비교/법인/산지/품종) 각 데이터 채움, 실시간 경매 가로 카드.
- **예측(`/prediction`)**: 조건 바(작물/품종/시장/출하량 ▼), 히어로 카드, 비교 카드 2개, 10일 예측 라인+신뢰구간+휴장 마커, 실거래 참조, 근거 접힘/펼침, 기술 신뢰 섹션. `defaultExpanded` 라우트도 그대로 유지.
- **판매처(`/sales`)**: 조건 바 + 현재가/예측가 토글, 추천 요약, 인사이트 배너, 시장 랭킹(클릭→상세 시트), 정렬 버튼, 계산 기준, 서브탭(시장 추천/채널 비교 BETA).
- **작물(`/crop`)**: 내 작물·관심 작물 가로 카드(클릭→상세 시트), 유망 작물 추천 리스트, 리스크vs수익성 버블 차트(recharts), 필터.

## 6. 문구·일관성 정리

- 앱 타이틀 "FarmInsight / 농산물 시세 예측"으로 통일
- placeholder/"--" 제거, 모든 가격 `원/단량` 명기
- 상승 빨강(#EF4444)/하락 파랑(#3B82F6) 토큰화
- 작물 칩은 `CropChip` 사용으로 강제 → 이모지 누락 불가
- 표 거래량 단위는 기존 메모리 규칙(숫자+t, k 금지) 유지

## 7. 예외 상태

`SkeletonCard`, `EmptyState`, `StaleBanner`(연노랑 #FEF3C7), `ForecastUnavailable`, `MarketHolidayNotice`, `NetworkError` 컴포넌트를 각 탭/시트에 배치.

## 기술 메모

- 새 의존성 없음(이미 recharts/shadcn drawer/dialog 보유 가정, 없으면 shadcn drawer만 추가)
- 라우팅은 `react-router-dom` 기존 구조 유지
- 모든 바텀시트는 `Drawer`(vaul) 기반 단일 컴포넌트로 추상화하여 호출부 일관성 확보
- mock 데이터 생성은 결정적(seed) 함수로 만들어 새로고침해도 같은 값 유지
